Integração com prontuários eletrônicos

O Risco Cirúrgico by YES Médicos pode ser integrado com qualquer prontuário eletrônico. A Integração é gratuita e oferecemos uma API endpoint para que seu prontuário a consuma. Sobre o Risco Cirúrgico

O Risco Med é um programa para emitir laudos de risco cirúrgico pré-operatório sem custo mensal ou fidelidade. 7 índices de risco cirúrgico estão disponíveis assim como análise de risco trombótico. Oferecemos planos especiais para grandes volumes (hospitais, prontos atendimentos, etc).

Se você é representante de hospitais ou prontos atendimentos, clique aqui para parcerias e planos especiais para grandes volumes.
